본 문서는 `Smarter Store API` 백엔드 프로젝트의 초기 설정 과정을 안내합니다.

## 1. 프로젝트 개요

- **목적**: TDD/DDD 방법론에 기반한 REST API 서버 구축
- **주요 기술 스택**:
  - **언어**: Kotlin
  - **프레임워크**: Spring Boot 3.3.0
  - **빌드 시스템**: Gradle (Kotlin DSL)
  - **데이터베이스**: PostgreSQL 18
  - **JDK**: Java 21

## 2. 사전 준비 (Prerequisites)

- **JDK 21**: [Oracle JDK](https://www.oracle.com/java/technologies/downloads/#jdk21-windows) 또는 [OpenJDK](https://jdk.java.net/21/)
- **PostgreSQL 18**: [공식 홈페이지](https://www.postgresql.org/download/)
- **IDE**: IntelliJ IDEA

## 3. 프로젝트 초기 설정 (요약)

1.  **Spring Initializr**: Gradle-Kotlin, Spring Boot 3.3.0, Java 21 선택.
2.  **의존성 추가**: Spring Web, JPA, PostgreSQL, Jackson-Kotlin.
3.  **DB 설정**: `application.yml`에 JDBC URL 및 계정 정보 설정.

## 4. TDD/DDD 기본 구조

-   **Domain**: `@Entity` 클래스 정의 (비즈니스 로직 포함).
-   **Repository**: `JpaRepository` 인터페이스 정의.
-   **Test**: `MockK`를 활용한 단위 테스트 작성.

## 5. 실행

`./gradlew bootRun` 명령으로 애플리케이션을 실행합니다.
