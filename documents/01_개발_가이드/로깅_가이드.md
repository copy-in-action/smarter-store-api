# 로깅 가이드 (Logging Guide)

## 개정이력

| 버전 | 일자 | 작성자 | 내용 |
|------|------|--------|------|
| 1.0 | 2025-12-29 | Gemini | 초기 로깅 정책 문서화 |

---

이 문서에서는 Smarter Store API 프로젝트의 로깅 정책, 설정 요약, 로거 사용 예시 및 외부 로그 경로 설정에 대해 설명합니다.

## 1. 주요 정책
- **파일 출력**: 콘솔 및 파일 동시 출력.
- **현재 로그**: `smarter-store.log`
- **롤링 정책**: 날짜 변경 또는 10MB 초과 시 아카이빙.
- **보관 기간**: 최대 30일.

## 2. 설정 요약 (`logback-spring.xml`)
- **`SizeAndTimeBasedRollingPolicy`**: 날짜/용량 복합 롤링.
- **로그 레벨**: Root `INFO`, 패키지 `DEBUG`.

## 3. 로거 사용 예시

```kotlin
private val logger = LoggerFactory.getLogger(javaClass)

fun doSomething() {
    logger.info("작업을 시작합니다...")
    logger.error("오류가 발생했습니다!", exception)
}
```

## 4. 환경별 로그 경로

*   **Local**: `./logs/smarter-store-local/` (프로젝트 내부)
*   **Prod**: `/home/cic/logs/smarter-store/` (서버 절대 경로)
